# LLM Benchmark Studio

Generate and compare landing pages created by different Large Language Models (OpenAI, Claude, Gemini) in real-time. Perfect for benchmarking LLM capabilities and quality.

## Features

- 🤖 **Multi-LLM Support** - Use OpenAI, Claude, or Gemini simultaneously
- ⚡ **Real-time Generation** - Stream landing page HTML as it's generated
- 📊 **Performance Metrics** - Track generation time for each model
- 🎨 **Side-by-Side Comparison** - View all generated pages in tabbed interface
- 🌐 **Professional UI** - Beautiful, responsive, animated interface
- 📱 **Mobile Ready** - Works perfectly on all devices

## Included

- `app.py` - Flask backend with multi-LLM support
- `templates/` - Professional HTML templates
- `static/` - CSS animations and interactive JavaScript
- `sample_outputs/` - Example landing pages generated by each LLM
- `Landing Page Generator with OpenAI, Claude, and Gemini.ipynb` - Original notebook
- `requirements.txt` - Python dependencies
- `.env.example` - Environment variable template

## Quick Start

1. **Create virtual environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure environment:**
   - Copy `.env.example` to `.env`
   - Add your API keys:
     - `OPENAI_API_KEY` (for OpenAI)
     - `OPENROUTER_API_KEY` (for Claude and Gemini via OpenRouter)

4. **Run the app:**
   ```bash
   python app.py
   ```

5. **Open in browser:**
   - Visit `http://127.0.0.1:5000`

## Usage

1. Enter a product or service description
2. Select which LLMs to benchmark (OpenAI, Claude, Gemini)
3. Click "Generate Pages"
4. Watch as landing pages are generated in real-time
5. Compare the results in the tabbed interface
6. Check generation times and performance metrics

## Environment Variables

- `OPENAI_API_KEY` - OpenAI API key (for GPT models)
- `OPENROUTER_API_KEY` - OpenRouter API key (for Claude and Gemini)
- `FLASK_SECRET_KEY` - Secret key for Flask sessions (recommended for production)

## Sample Outputs

The `sample_outputs/` folder contains example landing pages generated by:

## Technologies

- **Backend**: Flask, OpenAI Python SDK, OpenRouter API
- **Frontend**: Bootstrap 5, Vanilla JavaScript, CSS Animations
- **Streaming**: Server-Sent Events (SSE) for real-time output
