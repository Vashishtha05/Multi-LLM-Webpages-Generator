# LLM Benchmark Studio

A professional Streamlit app that generates landing pages using multiple LLMs (OpenAI, Claude, Gemini) and lets you compare results side-by-side with live previews.

## Features

- **Multi-LLM Support** — OpenAI GPT-4o-mini, Claude 3.5 Sonnet, Gemini 2.0 Flash
- **Live HTML Preview** — Rendered landing pages in embedded iframes
- **Side-by-Side Comparison** — Tabbed interface to switch between model outputs
- **Performance Metrics** — Generation time and output size per model
- **Max Tokens Slider** — Control response length (500–4000)
- **Download HTML** — One-click download for each generated page
- **View Source** — Expandable code viewer for generated HTML
- **Animated UI** — Gradient background, glass-morphism cards, polished layout

## Included

- `app.py` — Streamlit app (all-in-one)
- `sample_outputs/` — Example landing pages generated by each LLM
- `Landing Page Generator with OpenAI, Claude, and Gemini.ipynb` — Original notebook
- `requirements.txt` — Python dependencies
- `.env.example` — Environment variable template

## Quick Start

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Configure environment:**
   - Copy `.env.example` to `.env`
   - Add your API keys

3. **Run the app:**
   ```bash
   streamlit run app.py
   ```

4. **Open in browser** — Streamlit will open automatically at `http://localhost:8501`

## Environment Variables

| Variable | Required | Purpose |
|---|---|---|
| `OPENAI_API_KEY` | Optional | OpenAI models directly |
| `OPENROUTER_API_KEY` | Optional | Claude & Gemini via OpenRouter |

At least one key is required.

## Usage

1. Enter a product/service description in the sidebar
2. Select which LLMs to benchmark
3. Adjust the Max Tokens slider
4. Click **Generate Pages**
5. Compare previews in the tabbed interface
6. Download any HTML you like
